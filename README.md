### AMTG

#### Installation:

``` sudo npm install gulp tsd -g ```

``` git clone git@github.com:OR13/AMTG.git ```

``` npm install && bower install && tsd install```


#### Local development:

``` gulp serve ```


#### Production build:

``` gulp build ```


#### Deploy build to Github Pages:

``` gulp gh-deploy ```


#### Recommended IDEs:

- [Atom](https://atom.io/)
- [Visual Studio Code](https://code.visualstudio.com/)
